# WinLab Backend

Modular FastAPI backend skeleton for the WinLab HR Super-App.

## Modules

- `identity`: users, employees, departments, roles, RBAC/ABAC, audit identity context
- `recruiting`: reused/adapted ai-recruiter domain
- `chatbot`: conversation orchestration runtime
- `onboarding`: adaptation plans, tasks, milestones
- `preboarding`: security questionnaires and employment documents
- `knowledge`: knowledge base and RAG content lifecycle
- `surveys`: pulse surveys, exit interviews, feedback campaigns
- `integrations`: 1C, Yandex Messenger, Sber Pulse adapters
- `analytics`: operational dashboards and metrics
